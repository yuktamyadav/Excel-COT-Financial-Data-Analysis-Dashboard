# 📊 COT Financial Data Analysis Dashboard

## 📌 Description

I've built a holistic dashboard, designed to streamline COT Data Analysis for F&O Traders globally. This COT Dashboard can be utilized to analyze key metrics that are crucial for a Trader to assess Active Contracts in the market, which gives them the ability to scrutinize their decision making while Trading or Hedging in the F&O markets. The Dashboard utilizes the power of Excel Macros, using which, I've Automated certain tasks in this Dashboard. I've also utilized the power of Generative AI that use latest LLMs, for this project. GenAI helped me to create different Macros codes that can be further edited on VBA, To automate ETL tasks for spreadsheet dataset.

## 🛠️ Tech Stack

The dashboard was built using the following tools and technologies:

- **Excel** – Main platform used for data analysis, report and dashboard building
- **Pivot Tables** – To analyse the data in a tabular form with advanced filters
- **Pivot Charts** – To visualize the data of Pivot Tables with advanced visual charts
- **Slicers** – Used to filter out data visualizations
- **Data Visualizations** – To visualize COT financial data across categories of insitutional investors, and for other related metrics
- **Macros VBA** – It are used to automate several tasks such pulling, cleaning and fixing the COT Data.
- **File Formats** – `.xlsx` for development and `.png` for dashboard previews

## 🗄️ Dataset

The data is pulled from various websites that host CFTC's data suite, using a semi-automated Macros script.

## 💡 Features / Highlights

### 💱 The Dashboard contains categorical data from the following types of COT Reports -

1. **Legacy COT Report** : Commercial Hedgers, Non Commercial Speculators or Investors, and Non Reportables (Retail Traders).
2. **TFF COT Report** : Dealer/Intermediaries, Institutional Asset Managers and Leveraged Funds.

### 📊 Key Data Metrics in the Datasets (With a Conditional Formatting) for Comprehensive Speculative Analysis across all categories of data -

1. Long Contracts
2. Short Contracts
3. Longs/Shorts Changed (+/-) from Last Week
4. Total Contracts
5. Percentage of Long/Short Contracts
6. Net Positions
7. Open Interest

### 📈 Analytical Visualizations -

1. Longs vs Shorts : Line Chart
2. Longs vs Shorts Changed (+/-) : Line Chart
3. Net Positions vs Open Interest : Clustered Column + Line Chart
4. Net Positions vs Non Reportable Net Positions : Clustered Column + Line Chart

### 📜 Macros Automation Codes added using Generative AI -

1. Addition/Deletion of Row across Sheets, To extract & load new COT data.
2. Error Correction of specific cells in a newly added row across Sheets.
3. Load COT Values on newly added rows.
